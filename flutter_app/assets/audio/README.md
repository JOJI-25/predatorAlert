# Siren Audio Placeholder

This directory should contain the siren audio file for predator alerts.

## Required Files

1. `siren.mp3` - Main siren alarm sound (should loop seamlessly)
2. `notification.mp3` - Short notification sound (optional)

## Recommended Sources

- [Freesound.org](https://freesound.org) - Search for "alert siren" or "emergency alarm"
- [Pixabay](https://pixabay.com/sound-effects/) - Free sound effects

## Audio Requirements

- Format: MP3 or WAV
- Duration: 2-5 seconds (will be looped)
- Volume: Normalized to -3dB
- Style: Industrial alarm or emergency siren

## Note

If no audio file is present, the app will attempt to use a fallback tone from an online source.
